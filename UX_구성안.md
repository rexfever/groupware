# 엔터프라이즈 포탈 그룹웨어 UX 구성안

## 1. 전체 레이아웃 구조

### 1.1 기본 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│  Header (고정)                                          │
│  [로고] [홈] [전자결제] [게시판] [임직원] [알림] [프로필] │
├──────────┬──────────────────────────────────────────────┤
│          │                                              │
│ Sidebar  │  Main Content Area                          │
│ (좌측)   │  (동적 콘텐츠)                               │
│          │                                              │
│ - 메뉴1  │                                              │
│ - 메뉴2  │                                              │
│ - 메뉴3  │                                              │
│          │                                              │
└──────────┴──────────────────────────────────────────────┘
```

### 1.2 반응형 브레이크포인트
- Desktop: 1200px 이상 (사이드바 + 메인 콘텐츠)
- Tablet: 768px ~ 1199px (접을 수 있는 사이드바)
- Mobile: 767px 이하 (햄버거 메뉴, 사이드바 오버레이)

## 2. 홈 화면 (Dashboard)

### 2.1 레이아웃
```
┌─────────────────────────────────────────────────────────┐
│  Header                                                  │
├──────────┬──────────────────────────────────────────────┤
│          │  ┌──────────────┐  ┌──────────────┐         │
│          │  │ 오늘의 일정   │  │ 미결재 문서   │         │
│          │  │              │  │              │         │
│ Sidebar  │  └──────────────┘  └──────────────┘         │
│          │  ┌──────────────┐  ┌──────────────┐         │
│          │  │ 최근 게시글   │  │ 공지사항      │         │
│          │  │              │  │              │         │
│          │  └──────────────┘  └──────────────┘         │
│          │  ┌──────────────┐  ┌──────────────┐         │
│          │  │ 빠른 메뉴     │  │ 조직도 미리보기│        │
│          │  │              │  │              │         │
│          │  └──────────────┘  └──────────────┘         │
└──────────┴──────────────────────────────────────────────┘
```

### 2.2 포틀릿 설계

#### 포틀릿 공통 요소
- 헤더: 제목 + 설정 아이콘 (⚙️) + 닫기 아이콘 (✕)
- 리프레시 버튼
- 드래그 핸들 (헤더 영역)
- 리사이즈 핸들 (우하단)

#### 포틀릿별 상세 설계

**1. 오늘의 일정 포틀릿**
- 크기: 2x2 그리드
- 내용:
  - 오늘 날짜 표시
  - 일정 목록 (시간, 제목, 장소)
  - "일정 추가" 버튼
  - "전체 일정 보기" 링크

**2. 미결재 문서 포틀릿**
- 크기: 2x2 그리드
- 내용:
  - 결제 대기 문서 수 배지
  - 문서 목록 (제목, 결제 단계, 기한)
  - "결제하기" 버튼
  - "전체 문서 보기" 링크

**3. 최근 게시글 포틀릿**
- 크기: 2x2 그리드
- 내용:
  - 게시판별 탭 또는 통합 목록
  - 게시글 제목, 작성자, 작성일
  - "더보기" 링크

**4. 공지사항 포틀릿**
- 크기: 2x2 그리드
- 내용:
  - 중요 공지 상단 고정
  - 공지 제목, 작성일, 중요도 표시
  - "전체 공지 보기" 링크

**5. 빠른 메뉴 포틀릿**
- 크기: 2x1 그리드
- 내용:
  - 아이콘 그리드 (3x3 또는 4x4)
  - 자주 사용하는 메뉴 바로가기
  - 커스터마이징 가능

**6. 조직도 미리보기 포틀릿**
- 크기: 2x2 그리드
- 내용:
  - 현재 사용자 부서 조직도 트리
  - 확대/축소 기능
  - "전체 조직도 보기" 링크

### 2.3 포틀릿 커스터마이징
- 설정 모달:
  - 포틀릿 추가/제거
  - 포틀릿 크기 조정
  - 포틀릿 순서 변경
  - 기본 레이아웃 저장/복원

## 3. 상단 메뉴 (Header)

### 3.1 구성 요소
```
┌─────────────────────────────────────────────────────────┐
│ [로고] [홈] [전자결제] [게시판] [임직원] [관리자] [🔔] [👤] [로그아웃] │
└─────────────────────────────────────────────────────────┘
```
*관리자 메뉴는 관리자 권한이 있는 사용자에게만 표시

### 3.2 상세 설계
- **로고 영역**: 클릭 시 홈으로 이동
- **메인 메뉴**: 
  - 호버 시 서브메뉴 드롭다운 (있는 경우)
  - 활성 메뉴 하이라이트
  - 모바일: 햄버거 메뉴로 변환
- **알림 아이콘**:
  - 미읽은 알림 수 배지
  - 클릭 시 알림 패널 열기
  - 알림 목록, 읽음 처리, 전체 삭제
- **프로필 영역**:
  - 프로필 사진 또는 아바타
  - 클릭 시 드롭다운: 내 정보, 설정, 로그아웃

### 3.3 스타일 가이드
- 높이: 64px
- 배경: 흰색 또는 브랜드 컬러
- 그림자: 하단에 subtle shadow
- 고정 위치: 스크롤 시에도 상단 고정

## 4. 좌측 사이드바 메뉴

### 4.1 구조
```
┌──────────┐
│ 📋 메뉴  │
├──────────┤
│ 📁 전자결제│
│   ├─ 작성하기│
│   ├─ 결제대기│
│   ├─ 진행중  │
│   └─ 완료   │
├──────────┤
│ 📝 게시판 │
│   ├─ 공지사항│
│   ├─ 자유게시판│
│   └─ 자료실 │
└──────────┘
```

### 4.2 동작 방식
- 상단 메뉴 선택 시 해당 섹션의 서브메뉴 표시
- 접기/펼치기 토글 버튼
- 현재 페이지 하이라이트
- 모바일: 오버레이 방식, 배경 어둡게 처리

### 4.3 스타일
- 너비: 240px (펼침) / 64px (접힘)
- 배경: 흰색 또는 연한 회색
- 아이콘 + 텍스트
- 호버 효과: 배경색 변경

## 5. 임직원 검색 화면

### 5.1 검색 영역
```
┌─────────────────────────────────────────┐
│  🔍 [검색어 입력...]  [고급검색 ▼]      │
│  필터: [부서 ▼] [직급 ▼] [전체]         │
└─────────────────────────────────────────┘
```

### 5.2 검색 결과
- **카드형 레이아웃** (기본)
  ```
  ┌──────────┐  ┌──────────┐  ┌──────────┐
  │ [사진]    │  │ [사진]    │  │ [사진]    │
  │ 이름      │  │ 이름      │  │ 이름      │
  │ 부서/직급 │  │ 부서/직급 │  │ 부서/직급 │
  │ [이메일]  │  │ [이메일]  │  │ [이메일]  │
  └──────────┘  └──────────┘  └──────────┘
  ```

- **리스트형 레이아웃** (선택 가능)
  ```
  ┌──────────────────────────────────────┐
  │ [사진] 이름 | 부서 | 직급 | 이메일    │
  ├──────────────────────────────────────┤
  │ [사진] 이름 | 부서 | 직급 | 이메일    │
  └──────────────────────────────────────┘
  ```

### 5.3 임직원 상세 정보 모달
- 프로필 사진 (큰 사이즈)
- 기본 정보 (이름, 사번, 부서, 직급)
- 연락처 (이메일, 전화번호, 내선번호)
- 조직도에서의 위치
- 액션 버튼: 이메일 보내기, 전화 걸기, 메시지 보내기

### 5.4 자동완성
- 검색어 입력 시 실시간 자동완성 드롭다운
- 최근 검색어 저장
- 인기 검색어 표시

### 5.5 AD 동기화 상태 표시
- 사용자 카드/리스트에 동기화 상태 아이콘 표시
- 최근 동기화 일시 표시
- 동기화 오류 시 경고 표시

## 6. 전자결제 화면

### 6.1 메인 화면 (좌측 메뉴 + 메인 영역)
```
좌측 메뉴:
- 작성하기
- 결제대기
- 진행중
- 완료
- 반려

메인 영역:
- 탭: 전체 / 품의서 / 지출결의서 / 휴가신청서
- 필터: 기간, 상태, 검색
- 문서 목록 테이블
```

### 6.2 문서 작성 화면
```
┌─────────────────────────────────────────┐
│  [← 뒤로]  문서 작성                     │
├─────────────────────────────────────────┤
│  문서 종류: [품의서 ▼]                  │
│  제목: [________________________]      │
│  내용: [________________________]       │
│         [________________________]       │
│  결제 라인:                              │
│    [추가] 1. 부서장 → 2. 본부장 → 3. 대표│
│  첨부파일: [파일 선택] [파일1.pdf] [✕]  │
│  [임시저장] [제출하기]                   │
└─────────────────────────────────────────┘
```

### 6.3 결제 프로세스
- **결제 대기 문서 상세**
  - 문서 내용 표시
  - 결제 라인 시각화 (진행 단계 하이라이트)
  - 결제 의견 입력
  - 승인/반려 버튼
  - 이전 결제자 의견 보기

- **결제 진행 현황**
  - 타임라인 형태로 결제 이력 표시
  - 각 단계별 상태 (대기/승인/반려)

## 8. 사용자 플로우

### 8.1 로그인 → 홈 화면
1. 로그인 페이지
2. 인증 성공 → 홈 화면 (포틀릿 표시)
3. 포틀릿에서 빠른 액션 가능

### 8.2 전자결제 작성 플로우
1. 상단 메뉴 "전자결제" 클릭
2. 좌측 메뉴 "작성하기" 선택
3. 문서 양식 선택
4. 내용 작성 및 결제 라인 설정
5. 제출 → 결제 프로세스 시작

### 8.3 게시글 작성 플로우
1. 상단 메뉴 "게시판" 클릭
2. 게시판 선택
3. "글쓰기" 버튼 클릭
4. 내용 작성 및 첨부
5. 등록 → 게시판 목록으로 이동

### 8.4 임직원 검색 플로우
1. 상단 메뉴 "임직원" 클릭
2. 검색어 입력 (자동완성)
3. 결과 확인
4. 프로필 클릭 → 상세 정보 모달
5. 연락처 액션 수행

### 8.5 관리자 플로우
1. 상단 메뉴 "관리자" 클릭 (관리자 권한 필요)
2. 좌측 메뉴에서 관리 항목 선택
3. 관리 작업 수행
4. 변경 사항 저장 및 확인

## 9. UI/UX 원칙

### 9.1 디자인 원칙
- **일관성**: 전체 시스템에서 일관된 디자인 언어
- **직관성**: 아이콘과 레이블을 명확하게
- **접근성**: 키보드 네비게이션, 스크린 리더 지원
- **반응성**: 빠른 피드백 (로딩, 성공, 에러)

### 9.2 컬러 시스템
- **Primary**: 브랜드 컬러 (예: #1890ff)
- **Success**: #52c41a
- **Warning**: #faad14
- **Error**: #f5222d
- **Neutral**: #8c8c8c
- **Background**: #f5f5f5

### 9.3 타이포그래피
- 제목: 24px, 20px, 18px
- 본문: 16px, 14px
- 작은 텍스트: 12px
- 폰트: 시스템 폰트 또는 웹 폰트 (Noto Sans KR)

### 9.4 간격 시스템
- 4px 기준 그리드
- 패딩: 8px, 16px, 24px, 32px
- 마진: 동일

### 9.5 애니메이션
- 페이지 전환: 200-300ms
- 호버 효과: 150ms
- 모달 등장: 300ms ease-out
- 부드러운 스크롤
